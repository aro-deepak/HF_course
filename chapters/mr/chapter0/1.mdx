# परिचय[[introduction]]

Hugging Face कोर्समध्ये तुमचे स्वागत आहे! हा परिचय तुम्हाला working environment सेट करण्यासाठी मार्गदर्शन करेल. जर तुम्ही हा कोर्स नुकताच सुरू करत असाल, तर आम्ही प्रथम [Chapter 1](/course/chapter1) पाहण्याची शिफारस करतो, त्यानंतर येथे परत येऊन तुमचे environment सेट करा जेणेकरून तुम्ही स्वतः कोड वापरून पाहू शकता.

या कोर्समध्ये आपण वापरणार असलेल्या सर्व libraries Python packages म्हणून उपलब्ध आहेत. त्यामुळे येथे आपण Python environment कसे सेट करायचे आणि तुम्हाला लागणाऱ्या specific libraries कशा install करायच्या ते पाहू.

आपण working environment सेट करण्याच्या दोन पद्धती पाहू: **Colab notebook वापरून** किंवा **Python virtual environment वापरून**. तुम्हाला सोयीची वाटणारी पद्धत निवडा. जर तुम्ही beginner असाल, तर Colab notebook वापरून सुरुवात करण्याची आम्ही जोरदार शिफारस करतो.

ध्यानात घ्या की या मार्गदर्शनात Windows system बद्दल माहिती दिलेली नाही. जर तुम्ही Windows वापरत असाल, तर Colab notebook वापरून पुढे जाण्याची शिफारस केली जाते. जर तुम्ही Linux distribution किंवा macOS वापरत असाल, तर येथे दिलेल्या दोन्ही पद्धतींपैकी कोणतीही वापरू शकता.

या कोर्सचा मोठा भाग वापरण्यासाठी Hugging Face account आवश्यक आहे. त्यामुळे आत्ताच एक account तयार करण्याची शिफारस केली जाते: [create an account](https://huggingface.co/join).

## Using a Google Colab notebook[[using-a-google-colab-notebook]]

Colab notebook वापरणे ही सर्वात सोपी setup पद्धत आहे; तुमच्या browser मध्ये notebook सुरू करा आणि लगेच coding सुरू करा!

जर तुम्ही Colab शी परिचित नसाल, तर प्रथम [introduction](https://colab.research.google.com/notebooks/intro.ipynb) पाहण्याची शिफारस करतो. Colab तुम्हाला GPUs किंवा TPUs सारखे accelerating hardware वापरण्याची सुविधा देते आणि लहान workloads साठी ते मोफत उपलब्ध आहे.

Colab मध्ये काम करण्याची सवय झाल्यावर, एक नवीन notebook तयार करा आणि setup सुरू करा:

<div class="flex justify-center">
<img src="https://huggingface.co/datasets/huggingface-course/documentation-images/resolve/main/en/chapter0/new_colab.png" alt="An empty colab notebook" width="80%"/>
</div>

पुढील पायरी म्हणजे या कोर्समध्ये वापरल्या जाणाऱ्या libraries install करणे. Installation साठी आपण `pip` वापरणार आहोत, जो Python चा package manager आहे. Notebook मध्ये system commands चालवण्यासाठी त्यापूर्वी `!` character वापरला जातो. त्यामुळे 🤗 Transformers library खालीलप्रमाणे install करू शकता:

```
!pip install transformers
```

Python runtime मध्ये import करून package योग्यरीत्या install झाले आहे की नाही ते तपासू शकता:

```
import transformers
```

<img src="https://huggingface.co/datasets/huggingface-course/documentation-images/resolve/main/en/chapter0/install.gif" alt="A gif showing the result of the two commands above: installation and import" width="80%"/>
</div>

हे 🤗 Transformers चे एक हलके version install करते. विशेषतः, कोणतेही specific machine learning frameworks (जसे PyTorch किंवा TensorFlow) install केलेले नसतात. कारण आपण library मधील अनेक features वापरणार आहोत, त्यामुळे development version install करण्याची शिफारस केली जाते. यात जवळजवळ सर्व वापरासाठी आवश्यक dependencies असतात:

```
!pip install transformers[sentencepiece]
```

यासाठी थोडा वेळ लागू शकतो, पण त्यानंतर तुम्ही उर्वरित कोर्ससाठी तयार असाल!

## Using a Python virtual environment[[using-a-python-virtual-environment]]

जर तुम्हाला Python virtual environment वापरायचा असेल, तर पहिली पायरी म्हणजे तुमच्या system वर Python install करणे. सुरुवात करण्यासाठी [this guide](https://realpython.com/installing-python/) पाहण्याची शिफारस केली जाते.

Python install झाल्यानंतर, तुम्ही terminal मध्ये Python commands चालवू शकता. पुढील पायऱ्यांपूर्वी ते योग्यरीत्या install झाले आहे की नाही हे तपासण्यासाठी खालील command वापरू शकता: `python --version`. यामुळे तुमच्या system वर उपलब्ध असलेली Python version दिसेल.

Terminal मध्ये `python --version` सारखा Python command चालवताना, त्या command ला चालवणारा program तुमच्या system वरील "main" Python installation असतो. आम्ही सुचवतो की या main installation मध्ये कोणतेही packages install करू नयेत आणि प्रत्येक application साठी वेगळे environment तयार करावे — यामुळे प्रत्येक application ला त्याचे स्वतंत्र dependencies आणि packages मिळतात आणि compatibility issues टाळता येतात.

Python मध्ये हे [*virtual environments*](https://docs.python.org/3/tutorial/venv.html) वापरून केले जाते. हे self-contained directory trees असतात ज्यामध्ये एक विशिष्ट Python version आणि application साठी लागणारे packages असतात. असे virtual environment तयार करण्यासाठी अनेक tools उपलब्ध आहेत, पण येथे आपण official Python package वापरणार आहोत ज्याचे नाव [`venv`](https://docs.python.org/3/library/venv.html#module-venv) आहे.

प्रथम, तुमचे application ज्या directory मध्ये ठेवायचे आहे ती directory तयार करा — उदाहरणार्थ, home directory मध्ये *transformers-course* नावाची directory तयार करू शकता:

```
mkdir ~/transformers-course
cd ~/transformers-course
```

या directory मध्ये जाऊन Python `venv` module वापरून virtual environment तयार करा:

```
python -m venv .env
```

आता तुमच्या रिकाम्या folder मध्ये *.env* नावाची directory तयार झालेली असेल:

```
ls -a
```

```out
.      ..    .env
```

`activate` आणि `deactivate` scripts वापरून तुम्ही virtual environment मध्ये प्रवेश करू शकता किंवा बाहेर पडू शकता:

```
# Activate the virtual environment
source .env/bin/activate

# Deactivate the virtual environment
deactivate
```

`which python` command चालवून environment activate झाले आहे का ते तपासू शकता. जर path virtual environment कडे दाखवत असेल, तर environment यशस्वीपणे activate झाले आहे:

```
which python
```

```out
/home/<user>/transformers-course/.env/bin/python
```

### Installing dependencies[[installing-dependencies]]

Google Colab वापरण्याच्या आधीच्या विभागाप्रमाणे, पुढे जाण्यासाठी आवश्यक packages install करणे आवश्यक आहे. पुन्हा एकदा, `pip` package manager वापरून 🤗 Transformers चे development version install करू शकता:

```
pip install "transformers[sentencepiece]"
```

आता तुमचे environment पूर्णपणे तयार झाले आहे आणि तुम्ही पुढे सुरू करू शकता!
