
# परिचय[[introduction]]

Hugging Face कोर्समध्ये आपले स्वागत आहे! हा परिचय तुम्हाला कार्यरत वातावरण (working environment) सेट करण्यासाठी मार्गदर्शन करेल. जर तुम्ही हा कोर्स नुकताच सुरू करत असाल, तर आम्ही प्रथम [Chapter 1](/course/chapter1) पाहण्याची शिफारस करतो, त्यानंतर येथे परत येऊन तुमचे वातावरण सेट करा जेणेकरून तुम्ही कोड स्वतः वापरून पाहू शकाल.

या कोर्समध्ये आपण वापरणाऱ्या सर्व लायब्ररी Python पॅकेजेस म्हणून उपलब्ध आहेत, त्यामुळे येथे आपण Python वातावरण कसे सेट करायचे आणि आवश्यक लायब्ररी कशा इन्स्टॉल करायच्या हे पाहणार आहोत.

आपले कार्यरत वातावरण सेट करण्यासाठी आपण दोन पद्धती पाहणार आहोत: Colab notebook वापरणे किंवा Python virtual environment वापरणे. तुमच्यासाठी जी पद्धत अधिक सोयीची वाटते ती निवडा. नवशिक्यांसाठी आम्ही Colab notebook वापरून सुरुवात करण्याची जोरदार शिफारस करतो.

लक्षात ठेवा की या मार्गदर्शकात Windows प्रणालीसाठी माहिती दिलेली नाही. जर तुम्ही Windows वापरत असाल, तर Colab notebook वापरणे आम्ही सुचवतो. जर तुम्ही Linux distribution किंवा macOS वापरत असाल, तर येथे दिलेल्या कोणत्याही पद्धतीचा वापर करू शकता.

या कोर्समधील बहुतेक भागांसाठी तुम्हाला Hugging Face खाते आवश्यक असेल. त्यामुळे आम्ही आत्ताच खाते तयार करण्याची शिफारस करतो: https://huggingface.co/join

## Google Colab notebook वापरणे[[using-a-google-colab-notebook]]

Colab notebook वापरणे ही सर्वात सोपी पद्धत आहे; तुमच्या ब्राउझरमध्ये notebook सुरू करा आणि थेट कोडिंग सुरू करा!

जर तुम्हाला Colab बद्दल माहिती नसेल, तर आम्ही प्रथम हा परिचय पाहण्याची शिफारस करतो:
https://colab.research.google.com/notebooks/intro.ipynb

Colab तुम्हाला GPUs किंवा TPUs सारखे hardware accelerators वापरण्याची सुविधा देते, आणि लहान workloads साठी ते विनामूल्य आहे.

Colab मध्ये काम करण्याची सवय झाल्यावर, एक नवीन notebook तयार करा आणि खालीलप्रमाणे सेटअप सुरू करा.

पुढील टप्पा म्हणजे या कोर्समध्ये वापरणाऱ्या लायब्ररी इन्स्टॉल करणे. इन्स्टॉलेशनसाठी आपण `pip` वापरणार आहोत, जो Python साठीचा package manager आहे.

Notebook मध्ये system commands चालवण्यासाठी त्याच्या आधी `!` हे चिन्ह वापरले जाते.

```
!pip install transformers
```

पॅकेज योग्यरित्या इन्स्टॉल झाले आहे की नाही हे तपासण्यासाठी ते Python runtime मध्ये import करून पाहू शकता:

```
import transformers
```

यामुळे 🤗 Transformers ची अतिशय हलकी आवृत्ती इन्स्टॉल होते. विशेषतः, कोणतेही machine learning frameworks (जसे PyTorch किंवा TensorFlow) यामध्ये इन्स्टॉल केलेले नसतात.

कारण आपण या लायब्ररीच्या अनेक वेगवेगळ्या वैशिष्ट्यांचा वापर करणार आहोत, त्यामुळे आम्ही development version इन्स्टॉल करण्याची शिफारस करतो.

```
!pip install transformers[sentencepiece]
```

यासाठी थोडा वेळ लागू शकतो, परंतु त्यानंतर तुम्ही उर्वरित कोर्ससाठी पूर्णपणे तयार असाल!

## Python virtual environment वापरणे[[using-a-python-virtual-environment]]

जर तुम्हाला Python virtual environment वापरायचे असेल, तर पहिला टप्पा म्हणजे तुमच्या प्रणालीवर Python इन्स्टॉल करणे.

सुरुवात करण्यासाठी हा मार्गदर्शक वापरा:
https://realpython.com/installing-python/

Python इन्स्टॉल झाल्यानंतर तुम्ही तुमच्या terminal मध्ये Python commands चालवू शकता.

ते योग्यरित्या इन्स्टॉल झाले आहे का हे तपासण्यासाठी:

`python --version`

यामुळे तुमच्या प्रणालीवर उपलब्ध असलेली Python ची आवृत्ती दिसेल.

Terminal मध्ये Python command चालवताना तो command चालवणारा Python हा तुमच्या प्रणालीवरील "main" Python समजा.

मुख्य installation मध्ये packages इन्स्टॉल न करता प्रत्येक application साठी वेगळे environment तयार करणे चांगले असते.

Python मध्ये हे virtual environments वापरून केले जाते:
https://docs.python.org/3/tutorial/venv.html

Virtual environment म्हणजे स्वतंत्र directory असते ज्यामध्ये Python installation आणि त्या application साठी आवश्यक packages असतात.

आपण Python चे अधिकृत `venv` module वापरणार आहोत:
https://docs.python.org/3/library/venv.html#module-venv

प्रथम directory तयार करा:

```
mkdir ~/transformers-course
cd ~/transformers-course
```

यानंतर virtual environment तयार करा:

```
python -m venv .env
```

आता `.env` नावाची directory तयार झालेली असेल:

```
ls -a
```

```
.      ..    .env
```

Virtual environment activate / deactivate करण्यासाठी:

```
source .env/bin/activate
deactivate
```

Environment सक्रिय झाले आहे की नाही तपासण्यासाठी:

```
which python
```

उदाहरण:

```
/home/<user>/transformers-course/.env/bin/python
```

### Dependencies इन्स्टॉल करणे[[installing-dependencies]]

आवश्यक packages इन्स्टॉल करण्यासाठी:

```

```
